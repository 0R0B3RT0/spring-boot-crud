# spring-boot-crud
Projeto de CRUD com Spring Boot

## Executar a aplicação
### Crie o banco de dados
```bash
createdb -U postgres spring-boot-crud
```
### Compilar
```bash
mvn clean install
```

## Stack
* Java 11
* Spring Boot
* Spring Data JPA
* Maven;
* Postgres;
* JUnit;
